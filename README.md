# miniCFD

## Intention
This repo is a minimum implementation of *Data Structures and Algorithms (DSA)* used in *Computational Fluid Dynamics (CFD)*.

## Build
```shell
git clone https://github.com/pvcStillInGradSchool/miniCFD.git
cd miniCFD
mkdir build
cd build
cmake -S .. -B .  # cmake 3.13.5+
cmake --build .
./test/mesh_test
```
